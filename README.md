# America In The World

An AP US History project website built with [Next.js](https://nextjs.org) and [Tailwind CSS](https://tailwindcss.com).

## Project

**Question:** Is the United States a better version of itself in 1945 than it was in 1898?

**Students:** Anthony, Arnav, Martin

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Fonts:** Georgia / Times New Roman (serif)
- **Language:** TypeScript

## Deploy on Vercel

The easiest way to deploy this app is to use the [Vercel Platform](https://vercel.com/new).
