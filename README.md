# America In The World

A Next.js website for the **America in the World** history project by Anthony, Arnav, and Martin.

## Project Question

> **Is the United States a better version of itself in 1945 than it was in 1898?**

**Answer:** Yes — due to improved quality of life, a stronger economy, and reduced corruption achieved through decades of progressive reform, global leadership, and democratic expansion.

## Features

- **Hero section** with animated thesis and patriotic star-field background
- **Historical Overview** — 5-sentence decade-by-decade timeline (1890s–1940s)
- **Six Historical Figures** with animated score bars, collapsible analysis, and historical thinking skill badges:
  - Andrew Carnegie (1890s) — Continuity over Time
  - William Randolph Hearst (1900s) — Comparison
  - Jeannette Rankin (1910s) — Causation
  - W.E.B. DuBois (1920s) — Comparison
  - Huey Long (1930s) — Comparison (free slot)
  - Franklin D. Roosevelt (1940s) — Change over Time
- **Summary Table** with scores and average
- **Historical Thinking Skills Legend**

## Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router)
- [Tailwind CSS v4](https://tailwindcss.com/)
- TypeScript

## Getting Started

```bash

```
